# 치킨 먹이 급여 시스템이 잘 동작하는지 로깅하는 서버
    - ./server 경로에 개발
    - 도커 및 도커 컴포즈로 관리
    - 엔드 포인트 : ketiict.com
    - 포트 : 37210
    - 라즈베리파이에서 동작 기록이 저장되고 표와 함께 시각화되어 볼 수 있다
    - 동작해야 하는 시간에 동작하지 않은 것에 대해서도 로깅해야 한다 (라즈베리파이가 외부적인 이유(전력, 누수 등)로 동작안할수도 있기 때문)
    - 동작해야 하는 시간은 라즈베리파이에서 서비스를 시작할때 config.json 파일에 지정된 시간을 서버로 송출하여 알 수 있다
    - 프론트 개발은 NextJS, Shadcn, Tailwind CSS 사용
